# DLSU CSO LEAP 2026 Website

A centralized web platform for managing the classes for LEAP 2026

Built using **React + Vite** for fast performance and scalable development.

---

## Overview

The DLSU CSO LEAP 2026 Website is designed to streamline coordination between the Council of Student Organizations (CSO), student orgs, and the student body. It provides a unified system for the database of LEAP classes for the entire university vision mission week.

---

## Features

- User Authentication (Students & Admins)
- Event Management & Registration
- LEAP Classes Directory
- Announcements System
- Admin Dashboard
- Responsive UI (Mobile & Desktop)

---

## 🛠️ Tech Stack

- **Frontend:** React
- **Build Tool:** Vite
- **Routing:** React Router
- **Styling:** CSS / Tailwind CSS
